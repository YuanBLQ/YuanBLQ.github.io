为了创建逻辑清晰，内容可重用的 Playbook，Playbook 支持 import 和 include 两种方法。

使用的时候，可能会发现用法很相似，但是 Ansible 执行的时候还是有区别的。

import * 进来的语句都是在 Playbook 解析的时候就预处理完了。

include * 进来的语句是在 Playbook 执行的时候才会执行。
