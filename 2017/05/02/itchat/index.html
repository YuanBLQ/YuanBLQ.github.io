itchat 微信机器人

发表于 2017-05-02 | 分类于 小练习 | 阅读次数

通过 itchat 模块和微软小冰的微信公众号(xiaoice-ms)打造自己的微信机器人

实现原理

  • 通过微信的 python 接口 itchat获取微信消息
  • 将微信消息转发给”小冰”
  • 获取”小冰”处理后的返回消息
  • 将返回消息返回给微信消息的发送人
