Django创建与链接数据库

发表于 2017-02-03 | 分类于 Django | 阅读次数

创建python虚拟环境并安装 Django 和 psycopg2 以及 PostgreSQL

  1. 在要创建 python 虚拟环境的地方使用命令行输入 python -m venv XXX（虚拟环境名称）

  2. 运行 Scripts 文件夹中的 activate.bat 进入虚拟环境

  3. 通过pip命令（pip install XXX）安装 Django 和 psycopg2（可用 pip freeze 查看当前虚拟环境已经安装的库）

  4. 下载安装 PostgreSQL

创建 Django 项目和第一个 app、更改 app 的数据库配置

  1. 调用 Scripts 文件夹中的 django-admin 程序新建一个项目（…/Scripts/django-admin startproject 项目名称）

  2. 在项目中创建第一个app（相当于一个功能）

  3. 打开“项目名称”文件夹中的 setting.py ，更改 DATABASES 字典，将 PostgreSQL 的配置信息告诉 Django，然后由 Django 管理数据库。

在 Django 项目的 app 中创建数据表

在刚才创建的 app 中的 model.py 中创建数据表所需信息

将创建的数据表写入 PostgreSQL

  1. 先将要写入数据库的 app 在 setting.py 中定义

  2. 将 app 的 model.py 的表单创建一个历史纪录，方便以后修改数据库

  3. 将表单结构写入 PostgreSQL 数据库
